# SEO Action Items — kpbc.ca

## Google Setup (Do First)
- [ ] **Google Search Console** — Go to https://search.google.com/search-console, add kpbc.ca, submit sitemap (https://kpbc.ca/sitemap.xml)
- [ ] **Google Analytics** — Go to https://analytics.google.com, create property for kpbc.ca, share tracking ID so it can be added to the site
- [ ] **Google Business Profile — Toronto** — Go to https://business.google.com, create listing for Toronto office
- [ ] **Google Business Profile — Kingston** — Create listing for Kingston, Jamaica office
- [ ] **Google Business Profile — Bridgetown** — Create listing for Bridgetown, Barbados office
- [ ] **Google Business Profile — Georgetown** — Create listing for Georgetown, Guyana office
- [ ] **Google Business Profile — Port of Spain** — Create listing for Port of Spain, Trinidad & Tobago office

## Bing Setup
- [ ] **Bing Webmaster Tools** — Go to https://www.bing.com/webmasters, add kpbc.ca, submit sitemap

## Content & Backlinks
- [ ] **LinkedIn company page** — Create or update with kpbc.ca link and all office locations
- [ ] **Jamaica business directories** — List KPBC on Jamaican business directories (e.g., FindYello.com, JamaicaYP.com)
- [ ] **Barbados business directories** — List on Barbados directories
- [ ] **Guyana business directories** — List on Guyana directories
- [ ] **Trinidad & Tobago business directories** — List on T&T directories
- [ ] **Canadian business directories** — List on Canadian directories (e.g., Yellow Pages, BBB)
- [ ] **Blog/case studies** — Add a blog or case studies page showcasing projects in each country (improves SEO significantly)

## Technical
- [ ] **Page speed test** — Run https://pagespeed.web.dev for kpbc.ca and fix any issues flagged
- [ ] **Rich results test** — Run https://search.google.com/test/rich-results for kpbc.ca to validate structured data
- [ ] **Change hosting password** — Your TMD password was shared in chat, change it in cPanel > Password & Security
- [ ] **Change Resend API key** — Regenerate at https://resend.com/api-keys since it was shared in chat

## Ongoing (Monthly)
- [ ] Check Google Search Console for indexing errors and fix them
- [ ] Review search queries in Search Console — see what people search to find you
- [ ] Monitor page speed — Google penalizes slow sites
- [ ] Add new content regularly — Google favors active sites
